# Instalar Visual Studio Code 

- Una computadora con los requisitos necesarios 
- Internet estable 
- Saber instalar un programa (conocimientos minimos en sistemas operativos) 
- Conocimientos minimos en ingles 

# Documentacion
## Mi calculadora 

- presionar start para hacer un calculo
- para hacer una operacion debemos empezar colocando numeros en el display 
- luego de finalizar la operacion presionamos "-" para obtener el resultado
